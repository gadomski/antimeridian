# Command line interface

The **antimeridian** CLI (command line interface) takes a GeoJSON file path as
its input, and outputs a fixed GeoJSON dictionary to standard output.

## Installation

Install with **pip**:

```shell
python -m pip install 'antimeridian[cli]'
```
